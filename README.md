# Jobeet
## Installing
1. git clone `git@github.com:Vlad6847/Homework-Symfony-Symfony.git`
2. cd Homework-Symfony-Symfony
3. make start
4. make php
5. composer install
6. bin/console d:make:m
7. bin/console d:f:l 
8. http://localhost

## Adding new category
1. make php
2. bin/console app:create-category <category_name>
